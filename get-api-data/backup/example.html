/icws/{sessionId}/configuration/password-policies

GET

/icws/{sessionId}/configuration/password-policies

Retrieves a list of passwordPolicies. The results can be filtered by specifying a query. All list-based configuration object resources will have a complex property that will serve as its identity. This property is called the configurationId and consists of a {id, displayName, uri}.

The following result limits are enforced by the server regardless of the value of the Range header field:

  • If no properties are included (see select parameter), then all of the items will be returned.
  • If any properties are included (see select parameter), then a maximum of 200 items will be returned in the result.
